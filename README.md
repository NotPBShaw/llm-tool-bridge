# Llm Tool Bridge

Validated model-to-tool dispatch with typed payloads.

![CI](https://github.com/NotPBShaw/llm-tool-bridge/actions/workflows/ci.yml/badge.svg)

TypeScript bridge that maps model tool calls to validated local handlers.

## Quickstart

```bash
npm install
npm test
```
